# Фінальний проект Фази 1: TypeScript Task Manager

## Опис

Повнофункціональний менеджер задач з повною типізацією TypeScript.

## Вимоги

### Функціонал:
- ✅ Створення задач
- ✅ Видалення задач
- ✅ Зміна статусу
- ✅ Фільтрація за статусом
- ✅ Пошук задач
- ✅ Сортування задач

### Технічні вимоги:
- Сувора типізація (strict mode)
- Використання interfaces
- Використання union types
- Типізовані функції
- Використання generics (базове)
- Без використання `any`

## Структура даних

```typescript
// Приблизна структура (буде деталізована пізніше)
interface Task {
  // ...
}

type TaskStatus = 'todo' | 'in-progress' | 'done';
```

## Критерії успіху

- [ ] Весь код типізований
- [ ] Код компілюється без помилок
- [ ] Всі функції працюють коректно
- [ ] Код перевірений ментором
- [ ] Додані коментарі до складних частин

**Статус:** Очікує завершення Частин A і B

**Очікуваний час:** 2-3 години
