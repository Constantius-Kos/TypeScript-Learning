# Финальный проект Фазы 1: TypeScript Task Manager

## Описание

Полнофункциональный менеджер задач с полной типизацией TypeScript.

## Требования

### Функционал:
- ✅ Создание задач
- ✅ Удаление задач
- ✅ Изменение статуса
- ✅ Фильтрация по статусу
- ✅ Поиск задач
- ✅ Сортировка задач

### Технические требования:
- Строгая типизация (strict mode)
- Использование interfaces
- Использование union types
- Типизированные функции
- Использование generics (базовое)
- Без использования `any`

## Структура данных

```typescript
// Примерная структура (будет детализирована позже)
interface Task {
  // ...
}

type TaskStatus = 'todo' | 'in-progress' | 'done';
```

## Критерии успеха

- [ ] Весь код типизирован
- [ ] Код компилируется без ошибок
- [ ] Все функции работают корректно
- [ ] Код проверен ментором
- [ ] Добавлены комментарии к сложным частям

**Статус:** Ожидает завершения Частей A и B

**Ожидаемое время:** 2-3 часа
