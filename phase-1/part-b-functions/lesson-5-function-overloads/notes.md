# Мои заметки — Урок 5: Function Overloads
### ❓ Что такое "сигнатура"?

**Сигнатура функции** — это её "удостоверение личности": описание того, что функция принимает и что возвращает, **без кода внутри**.

```typescript
// Это СИГНАТУРА — только описание, без тела {}
function greet(name: string): string;

// Это РЕАЛИЗАЦИЯ — есть тело {}
function greet(name: string): string {
  return "Hello, " + name;
}
```

Сигнатура отвечает на три вопроса:
1. Как функция называется?
2. Какие параметры и каких типов принимает?
3. Что возвращает?

Слово пришло из математики — там "сигнатура" это тоже описание функции без её вычисления.
