# Мої нотатки — Урок 5: Function Overloads
### ❓ Що таке "сигнатура"?

**Сигнатура функції** — це її "посвідчення особи": опис того, що функція приймає і що повертає, **без коду всередині**.

```typescript
// Це СИГНАТУРА — тільки опис, без тіла {}
function greet(name: string): string;

// Це РЕАЛІЗАЦІЯ — є тіло {}
function greet(name: string): string {
  return "Hello, " + name;
}
```

Сигнатура відповідає на три питання:
1. Як функція називається?
2. Які параметри і яких типів приймає?
3. Що повертає?

Слово прийшло з математики — там "сигнатура" це теж опис функції без її обчислення.
