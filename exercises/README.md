# Ежедневные упражнения

Здесь будут храниться все практические задачи и упражнения.

## Структура

Упражнения организованы по темам:
- `primitives/` - Задачи на примитивные типы
- `arrays/` - Задачи на массивы
- `functions/` - Задачи на функции
- `mixed/` - Смешанные задачи

## Рекомендации

- Старайся решать минимум 1-2 задачи в день
- Не подглядывай в решения сразу - попробуй сам
- Если застрял больше 20 минут - попроси подсказку (`!hint`)
- После решения попроси ментора проверить код (`!review`)
