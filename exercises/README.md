# Щоденні вправи

Тут зберігатимуться всі практичні задачі та вправи.

## Структура

Вправи організовані за темами:
- `primitives/` - Задачі на примітивні типи
- `arrays/` - Задачі на масиви
- `functions/` - Задачі на функції
- `mixed/` - Змішані задачі

## Рекомендації

- Намагайся вирішувати мінімум 1-2 задачі на день
- Не підглядай у рішення одразу - спробуй сам
- Якщо застряг більше 20 хвилин - попроси підказку (`!hint`)
- Після вирішення попроси ментора перевірити код (`!review`)
