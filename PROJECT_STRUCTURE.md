## 📋 Опис ключових файлів

### 🔴 Критично важливі:

**`.cursorrules`**
- Повні інструкції для AI ментора
- Методологія навчання
- Структура уроків
- Команди для взаємодії
- ⚠️ НЕ ВИДАЛЯЙ І НЕ ЗМІНЮЙ без розуміння!

**`START_HERE.md`**
- Стартовий промпт для першої сесії
- Команди які можна використовувати
- Що очікувати від першого уроку

### 🟢 Важливі:

**`README.md`**
- Автоматично оновлюється ментором
- Показує твій прогрес
- Статистика навчання
- Історія досягнень

**`SETUP.md`**
- Покрокова інструкція налаштування
- Встановлення залежностей
- Перевірка працездатності
- Корисні команди

**`tsconfig.json`**
- Конфігурація компілятора TypeScript
- Увімкнено суворий режим (strict mode)
- Налаштовані шляхи для компіляції

**`package.json`**
- Залежності проекту (TypeScript)
- NPM скрипти для компіляції

## 🚀 Що робити далі?

### Крок 1: Налаштування
```bash
cd TypeScript
npm install
```

### Крок 2: Відкрий в IDE
Відкрий всю папку `TypeScript/` в Antigravity IDE

### Крок 3: Почни навчання
1. Відкрий `START_HERE.md`
2. Скопіюй стартовий промпт
3. Надішли його в Antigravity Agent
4. Слідуй інструкціям ментора

---

## 💡 Корисні поради

✅ **Роби:**
- Регулярно робити коміти прогресу в Git
- Записуй нотатки в `notes/my-learnings.md`
- Використовуй команди (`!hint`, `!review`, `!practice`)
- Проси пояснень коли незрозуміло

❌ **Не роби:**
- Не змінюй `.cursorrules` без необхідності
- Не видаляй структуру папок
- Не копіюй код бездумно
- Не поспішай - якість важливіша за швидкість

---

## 📞 Команди для AI ментора

**Під час навчання:**
- `!start` - розпочати перший урок
- `!hint` - попросити підказку
- `!solution` - показати рішення
- `!review` - перевірити код
- `!progress` - показати статистику
- `!next` - перейти до наступної теми

**Повний список команд** - у файлі `.cursorrules`
