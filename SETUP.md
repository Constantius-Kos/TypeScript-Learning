# 🛠️ Інструкція з налаштування проекту

## Крок 1: Встановлення залежностей

Відкрий термінал у папці проекту та виконай:

```bash
npm install
```

Це встановить TypeScript локально для проекту.

---

## Крок 2: Перевірка встановлення

Перевір версію TypeScript:

```bash
npx tsc --version
```

Має з'явитися версія TypeScript (наприклад, `Version 5.7.2`).

---

## Крок 3: Налаштування VS Code (необов'язково, але рекомендується)

### Встанови розширення:
1. **TypeScript** (зазвичай вбудоване)
2. **ESLint** (для перевірки коду)
3. **Error Lens** (для зручної підсвітки помилок)

### Увімкни автозбереження:
- `File > Auto Save` (постав галочку)

---

## Крок 4: Перший тест

Створи файл `test.ts` з вмістом:

```typescript
const message: string = "Hello, TypeScript!";
console.log(message);
```

Скомпілюй:

```bash
npx tsc test.ts
```

Запусти:

```bash
node test.js
```

Має вивести: `Hello, TypeScript!`

---

## Крок 5: Почни навчання

Тепер ти готовий! Відкрий файл `START_HERE.md` і слідуй інструкціям.

---

## Корисні команди

```bash
# Скомпілювати всі .ts файли
npm run build

# Запустити компіляцію в режимі відстеження
npm run watch

# Скомпілювати конкретний файл
npx tsc path/to/file.ts
```

## Що робити якщо щось не працює?

### TypeScript не компілюється:
1. Перевір встановлення: `npx tsc --version`
2. Переконайся що файл має розширення `.ts`
3. Перевір помилки в терміналі

### Помилки у VS Code:
1. Перезапусти VS Code
2. Відкрий Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
3. Введи: `TypeScript: Restart TS server`
