### Питання: extends в дженеріках — розширення чи обмеження?
`extends` в generic constraints читати як "є підтипом" / "відповідає формі".
- В класах: `extends` = взяти все від батька
- В дженеріках: `extends` = T повинен мати хоча б ці властивості

```ts
T extends { length: number }
// = "T є типом який має length"
// string ✅, number[] ✅, number ❌
```
