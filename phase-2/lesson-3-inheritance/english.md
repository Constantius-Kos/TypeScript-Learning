# Англійські слова — Урок 3: Наслідування

## typo *(тайпоу)*
**Переклад:** друкарська помилка, описка

**Форми:**
- є: *there is a typo* *(тайпоу)*
- була: *there was a typo* *(воз е тайпоу)*
- буде: *there will be a typo* *(віл бі е тайпоу)*

**В коді/помилках:**
- `"typo in variable name"` — помилка в назві змінної
- в code review: `"typo: dors → doors"`

**Кейси використання:**
- `"Fix typo in README"` — назва коміту
- `"Sorry, typo"` — в чаті під час code review
- `"Typo in function name"` — коментар в PR
